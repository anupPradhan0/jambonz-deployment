# jambonz-deployment

Deployment scaffold for a self-hosted [jambonz](https://www.jambonz.org/) stack (API server, feature server, webapp, SBC inbound/outbound, SIP sidecar) using Docker Compose.

## Quick start

```bash
cp .env.example .env
docker compose up -d --build
```
